What does it do?
----------------

This extension provides the basic setup for Page type related Information views.
It is not recommended to use this extension alone, as it only provides a basic
setup. In order to perform information view, an addition in a separate extension
is required.
Examples can be found in the following FGTCLB extensions:

* `Academic Studies`_

.. _Academic Studies: https://extensions.typo3.org/extension/academic_studies
